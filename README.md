# AnonyArmour

AnonyArmour is a modular, high-anonymity browser extension designed to block invasive third-party telemetry and actively spoof device fingerprinting attributes. It is optimized to neutralize enterprise-grade tracking platforms, document scanning OCR, and biometric selfie scanners.

This project is structured as a cross-browser extension supporting **Chrome/Brave** and **Firefox** from a single source directory.

---

## Directory Structure

*   **`src/`**: Contains common source files (`background.js`, `content.js`, `inject.js`, `popup.html`, `popup.js`).
*   **`manifests/`**: Contains browser-specific manifest configurations (`chrome/manifest.json`, `firefox/manifest.json`).
*   **`helpers/`**: Contains lazy scripts for creating desktop/browser-extension-page shortcuts.
*   **`dist/`**: The compiled extension output folders generated by the build script (`chrome/`, `firefox/`).
*   **`build.sh`**: Compilation helper script.

---

## How to Build

Run the compilation script in your terminal to assemble the browser-specific builds into the `dist/` directory:

```bash
chmod +x build.sh
./build.sh
```

---

## Installation

### A. Chrome / Brave / Chromium Browsers
1.  Open Brave/Chrome and navigate to `brave://extensions/` (or `chrome://extensions/`).
2.  Enable **Developer mode** (toggle in the top-right corner).
3.  Click **Load unpacked** (top-left corner).
4.  Select the **`dist/chrome`** directory.

### B. Firefox
1.  Open Firefox and navigate to `about:debugging#/runtime/this-firefox`.
2.  Click **Load Temporary Add-on...**
3.  Select any file (like `manifest.json`) inside the **`dist/firefox`** directory.

## Shortcut Helper Scripts (`helpers/`)

To make loading the extension page easier, you can run the lazy scripts provided inside the `helpers/` folder depending on your OS:

*   **Windows**: Right-click `create_shortcut.ps1` and select **Run with PowerShell** to create a Desktop shortcut directly pointing to `brave://extensions`.
*   **macOS**: Run `./create_shortcut_mac.sh` to execute the AppleScript that automatically fires up Brave's extension page.
*   **Linux**: Run `./create_shortcut_linux.sh` to build and register a `.desktop` launcher entry on your Desktop environment.

## Color-Coded Security Indicators

The extension popup features a live status banner detailing your security state:

*   🟢 **SYSTEM SECURE (Green)**: The environment is clean; no tracking attempts have occurred.
*   🟠 **ATTACKED - SECURED (Orange)**: Telemetry servers attempted to profile you, but the extension successfully returned fake/spoofed attributes.
*   🔴 **ATTACKED - DATA EXFILTRATED (Red)**: A fingerprinting attempt occurred while one or more shield modules were disabled, meaning raw hardware/system attributes were exposed.

## Local Auditing & Reporting

AnonyArmour tracks every probe and blocking event locally in your browser storage.
*   Click **Export Local Audit Report** in the extension popup to download a detailed markdown/text analysis (`AnonyArmour_Audit_Report_[Date].txt`) containing timestamps, origin URLs, exact probe targets, and shield status.
*   Click **Reset Counter & Logs** to purge logs from local memory.
